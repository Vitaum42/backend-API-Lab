# 🚀 Backend API Lab

Coleção de projetos de APIs back-end desenvolvidos com Java e Python, com foco em aprendizado, prática e evolução contínua.

---

## 📌 Sobre

Este repositório reúne implementações de APIs com o objetivo de:

* Consolidar conhecimentos em desenvolvimento back-end
* Praticar conceitos utilizados no mercado
* Evoluir através de projetos reais e progressivos

Aqui você encontrará diferentes abordagens, tecnologias e níveis de complexidade ao longo do tempo.

---

## 🛠️ Tecnologias

* Java (Spring Boot)
* Python (FastAPI / Flask)
* APIs REST
* Git e GitHub

Construir uma base sólida em desenvolvimento back-end, entendendo desde conceitos básicos até práticas mais avançadas utilizadas em aplicações reais.

# 💻 Comandos Utilizados (Bash)

## 📁 Criar e acessar diretório

```bash
mkdir nome-do-projeto
cd nome-do-projeto
```

## 🧪 Criar ambiente virtual

```bash
python -m venv venv
```

## ▶️ Ativar ambiente virtual

### Windows

```bash
venv\Scripts\activate
```

### Linux/Mac

```bash
source venv/bin/activate
```

## ⬆️ Atualizar pip

```bash
pip install --upgrade pip
```

## 📦 Instalar dependências

```bash
pip install fastapi uvicorn
```

## ▶️ Rodar servidor

```bash
uvicorn main:app --reload
```

# 🔄 Métodos HTTP

## 🟢 GET

Buscar dados sem alterar o estado da aplicação.

## 🔵 POST

Criar novos dados enviando informações no corpo da requisição.

## 🟡 PUT

Atualizar completamente um recurso existente.

## 🟠 PATCH

Atualizar parcialmente um recurso.

## 🔴 DELETE

Remover um recurso existente.
